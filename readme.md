# Challenge from GoStack Bootcamp

Foobar is a Python library for dealing with word pluralization.

## Installation

```bash
yarn install
```

## Usage

```bash
yarn dev
```

## Routes and port

Port is 3333

Routes:

POST /projects
POST /projects/:id/tasks
GET /projects
PUT /projects/:id
DELETE /projects/:id
