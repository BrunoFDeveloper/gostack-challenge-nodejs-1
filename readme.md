# Challenge from GoStack Bootcamp

This challenge is for make a Server with express with some rules.

## Installation

```bash
yarn install
```

## Usage

```bash
yarn dev
```

## Routes and port

Port is 3333

Routes:

POST /projects  
POST /projects/:id/tasks  
GET /projects  
PUT /projects/:id  
DELETE /projects/:id
